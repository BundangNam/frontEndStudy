jQuery Traversing(= DOM tree)

선택한 요소를 기준으로 위, 아래, 옆으로 이동하면서 선택

jQuery Ancestors - 상위객체

  • parent() - 선택한 요소의 부모객체(상위객체) 선택
  • parents() - 선택한 요소의 상위객체 모두 선택
  • parentsUntil() - 선택한 요소와 ()요소 사이의 상위객체 선택

jQuery Descendants - 하위객체

find() - 선택한 요소의 하위객체 중 ()객체
children() - 선택한 요소의 직접적인 하위객체

jQuery Siblings - 형제객체

  1. siblings() - 선택한 요소의 형제 객체
  2. next() - 선택한 요소의 형제 객체 중 다음객체
  3. nextAll() - 선택한 요소의 형제 객체 중 다음객체 전부
  4. nextUntil() - 선택한 요소의 형제 객체 중 ()요소 사이의 다음객체
  5. prev() - 선택한 요소의 형제 객체 중 이전객체
  6. prevAll() - 선택한 요소의 형제 객체 중 이전객체 전부
  7. prevUntil() - 선택한 요소의 형제 객체 중 ()요소 사이의 이전객체
