animate()

스크립트로 움직이는 동작을 적용

$('선택자').animate({ 스타일 }, 속도, easing, 콜백함수);

*스타일 - 속성:'값'

  • 여백 : margin / padding
  • 크기 : width / height
  • 위치 : top / right / bottom / left
  • 테두리 : border
  • 투명도 : opacity
  • 글자 : fontSize / lineHeight

속도

duration 애니메이션 효과를 완료할 때까지 걸리는 시간

milliseconds(1초=1000)

400(기본값) / fast(200) / slow(600)

easing

timing-function

'swing' : 시작과 끝부분은 천천히 중간은 빠르게(기본값)

'linear' : 일정한 속도

콜백함수

animate() 동작이 끝난 후 실행되어질 값
