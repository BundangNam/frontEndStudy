justify-content 좌우정렬
flex-start : 좌측정렬(기본값)
center : 가운데 정렬
flex-end : 우측정렬
space-between : 요소들 사이에 동일한 간격의 여백
space-around : 요소들 주변에 동일한 간격의 여백
align-items 상하정렬
strech : 요소들을 늘림(기본값)
flex-start : 상단정렬
cente : 가운데정렬
flex-end : 하단정렬
baseline : 상단정렬(글자의 하단기준)
flex-direction 정렬방향
row : 가로(수평)정렬(기본값) - 왼쪽에서 오른쪽으로
row-reverse : 가로(수평)정렬(기본값) - 오른쪽에서 왼쪽으로
column : 세로(수직)정렬 - 위에서 아래로
column-reverse : 세로(수직)정렬 - 아래에서 위로
order 순서지정
기본값은 0
order의 값은 숫자로 지정(양수,음수)
flex-wrap 묶음
nowrap : 요소들을 모두 한 줄에 정렬(기본값)
wrap : 요소들을 여러 줄에 정렬
wrap-reverse : 요소들을 여러 줄에 반대로(상하) 정렬
align-content 상하정렬(여러 줄)
strech : 영역에 맞게 늘림(기본값)
