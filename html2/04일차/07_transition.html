transition

동작과 함께 움직이는 효과를 적용

transition:property duration timing-function delay;

*property 속성

  • 위치 - top / right / left / bottom
  • 변형 - transform(translate / scale / rotate / skew)
  • 크기 - width / height
  • 색상 - color / background-color
  • 테두리 - border
  • 투명도 - opacity

*duration 재생시간

초단위를 사용(2초-2s)

배경색:노-보/가로:200-600/세로:200-400/테두리:5 오렌지
