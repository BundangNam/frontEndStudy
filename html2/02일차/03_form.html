form 폼

웹 페이지의 정보를 다른페이지로 전송하는 역할

action - 폼 내부의 데이터를 보내는 목적지 주소 지정

method - 폼을 서버로 전송하는 http방식을 지정(post / get)

target - action에서 지정한 스크립트파일이 현재나 다른위치에서 열리도록 지정(_self / _blank)

autocomplete - 양식의 자동완성기능(on/off)

  • 입력필드 - 글자를 입력하는 양식 :
  • 비밀번호 - 글자 입력시 *나.과 같은 기호로 대신 쓰이는 양식 :
  • 체크박스 - 복수선택 :
  • 라디오버튼 - 단일선택 :
  • 선택 - 드롭다운리스트 :
  • 다중입력필드 - 여러줄의 글자를 입력할 수 있는 양식:
  • 이메일 - 이메일 주소를 입력하는 유형 :
  • 전화번호 - 번호를 입력하는 유형 :
